# Arquitectura del sistema en la nube

Este sistema en la nube permite gestionar archivos corporativos y usuarios desde una aplicación web.

## Componentes
- Frontend web
- API REST
- Base de datos

## Seguridad
Se definen políticas base de acceso, respaldo y auditoría.
Se documentan controles operativos para el monitoreo del sistema.
Se incorpora cifrado AES-256 para la protección de datos en reposo.
Se añade TLS 1.3 para proteger la comunicación entre servicios.
Se utiliza un gestor de llaves para la administración criptográfica.
Se registran eventos críticos para auditoría de accesos y cambios.
